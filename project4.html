Pizza Sales Report

Pizza Sales Dashboard using SQL and Power BI

This project presents a comprehensive analysis of pizza sales data to uncover business insights and optimize sales strategies. Data was cleaned and aggregated using SQL, and key performance indicators (KPIs) were visualized using Power BI for better decision-making.

Step 1: SQL queries were used to calculate core KPIs — Total Revenue, Average Order Value, Total Pizzas Sold, and Total Orders.
Step 2: The dataset was then visualized in Power BI, showing patterns like order trends by day and month, category-based performance, and revenue by pizza size.

Technologies Used: SQL, Power BI, Data Cleaning, Data Visualization, Business Intelligence
