Call Analysis Dashboard

Call Center Performance Analysis using Excel

This project focuses on analyzing call center performance metrics using Microsoft Excel. The dashboard presents a detailed overview of key metrics such as the number of calls, total amount generated, average rating, and customer satisfaction trends.

Step 1: Raw call logs were cleaned and summarized using Excel formulas and pivot tables.
Step 2: A visually appealing dashboard was built with charts and KPIs to represent Call Trends, Duration Analysis, Rating Distribution, and Regional Performance.

Technologies Used: Microsoft Excel, Data Cleaning, Pivot Tables, Data Visualization

The insights from this dashboard help identify the most active regions, performance trends, and improvement opportunities in customer engagement.
